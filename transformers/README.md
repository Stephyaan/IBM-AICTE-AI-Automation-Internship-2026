# AI Project Manager Agent using n8n
> IBM AICTE AI Automation and Intelligent Solutions – 6-Week Summer Internship 2026

## Overview

This project demonstrates a multi-agent AI workflow built using n8n. The system automates project planning by analyzing a project description, generating milestones, assigning tasks based on team members' skills, tracking project progress, and updating a task tracker automatically.

Instead of manually managing projects, two AI agents collaborate to perform planning and execution tasks.

## Problem Statement

Project managers spend significant time:

- Breaking projects into milestones
- Creating detailed task lists
- Assigning work to team members
- Tracking project progress
- Sending reminders
- Updating task trackers

Manual project management is time-consuming, repetitive, and prone to delays.

## Solution

The AI Project Manager Agent automates the entire workflow using two AI agents.

### Planner Agent

Responsible for:

- Understanding the project
- Creating milestones
- Estimating timelines
- Planning execution

### Task Agent

Responsible for:

- Creating detailed tasks
- Assigning work
- Updating task sheets
- Tracking completion

## Workflow

1. User provides project details.
2. Read team member information from Excel.
3. Planner Agent creates milestones.
4. Planner sheet is updated.
5. Task Agent creates detailed tasks.
6. Task sheet is updated.
