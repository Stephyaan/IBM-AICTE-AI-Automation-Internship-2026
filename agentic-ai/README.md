# Auto-Reply to Incoming Emails using Relay.app

## Overview
This project demonstrates an AI-powered email automation workflow built using Relay.app. The workflow automatically classifies incoming Gmail emails using AI and performs actions based on the detected category.
The reusable classification logic has been modularized as a Relay Sequence, making it easier to maintain and reuse across multiple workflows.

## Sequence Logic

The workflow has been converted into a reusable Relay Sequence.

Sequence Steps

1. Receive input from workflow
2. Classify incoming email using AI
3. Check email category
4. If category is Personal Email
5. Send WhatsApp notification
6. Return output to workflow
